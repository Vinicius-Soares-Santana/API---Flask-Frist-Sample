# API---Flask-Frist-Sample
Simple REST API created for study of API development using FLASK in Python.

Project also uses SQLalchemy and Psycopg2 and connects to a database to retrieve all information available.

How to use:
install Flask, sqlalchemy, psycopg2, python in the code's file
Create a SQL Database
Replace the connection information in the line 23 "conn = psycopg2.connect(host = "127.0.0.1", database = "guide", user = "postgres", password="1234")" for your database information.
Replace the name of the database in the SQL Query in line 25
Run the code and open link to the local API
use endpoint "/all" to retrieve information.

code created and tested using VS code.
